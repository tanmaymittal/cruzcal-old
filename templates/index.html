You need to input your class numbers in the class number box.
This is different from the number followed by the class name.
For example we have the class CSE 115A. 115A is not the class number.
The class number for CSE 115A is 50444.
You can find out your class number by going to My Schedule in my.ucsc.edu
Input class numbers for all the classes you want to your calendar and press Fetch Information.
After that you should see a download button. Simply download the calendar file and import to your preferred calendar!
Current functionality only for Spring 2022.
